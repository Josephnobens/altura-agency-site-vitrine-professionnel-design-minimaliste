# Altura Agency Site Vitrine Professionnel   Design Minimaliste

## Vue d'ensemble
Site vitrine professionnel pour Altura Agency avec design minimaliste en une seule page. Utilise un dégradé bleu marine (#0D1B2A vers #1B263B) comme arrière-plan principal avec accents dorés (#FFD700). Navigation épurée avec sections compactes et espacement équilibré pour un look élégant et moderne.

## Structure de Navigation
- Navigation en une seule page avec sections défilantes
- En-tête avec texte stylé "ALTURA AGENCY" :
  - "ALTURA" en blanc (#FFFFFF) sans arrière-plan
  - "AGENCY" avec arrière-plan doré premium (#FFD700), texte en bleu marine (#0D1B2A), padding subtil et coins arrondis pour un effet premium
  - Texte en gras, élégant et centré verticalement avec espacement équilibré entre les deux mots
- Menu de navigation fixe minimaliste avec boutons épurés :
  - Accueil
  - Services
  - Portfolio
  - Contact
- Boutons de menu avec texte blanc par défaut
- Effet de survol avec transition subtile vers la couleur dorée (#FFD700)
- Transitions fluides entre les sections au scroll
- Équilibre esthétique entre le texte de l'en-tête et les boutons de navigation
- Séparateurs horizontaux fins dorés (#FFD700) entre chaque section majeure pour délimiter visuellement les zones
